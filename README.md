# ARP-Recherche-Non-Inform

Travaux pratiques du cours Algorithmes pour la Résolution de Problèmes (L3).
Le dépôt regroupe les modèles d’état, algorithmes de recherche non informée et informée, ainsi que leurs implémentations Java.

## Authors

Erkin Tunc Boya · Semih DOYNUK · Karam ELNASORY

Université Clermont Auvergne

problème des cruches, tour du cavalier, missionnaires et cannibales.
